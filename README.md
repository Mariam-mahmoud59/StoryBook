## 📖 About

**Story Book Creator** is an interactive, beautifully animated mobile application tailored for creating and reading delightful children's stories. Built with Flutter, it focuses on delivering a rich, visually stunning, and highly engaging user experience for both children and parents alike.

The application allows users to interact with stories through dynamic content, bringing imagination to life on the screen.

## ✨ Key Features

- **Read & Explore:** Dive into an immersive world of stories with beautiful illustrations.
- **Engaging UI & Animations:** Fluid, captivating transitions built using `flutter_animate` that keep children engaged.
- **Child-Friendly Typography:** Clean, readable typography powered by Google Fonts.
- **Cross-Platform:** Available and optimized for both Android and iOS devices.
- **Responsive Layout:** Adjusts perfectly to everything from mobile phones to larger tablet screens.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Ensure you have the following installed to run the project:
* [Flutter SDK](https://docs.flutter.dev/get-started/install) (version `>=3.0.0 <4.0.0`)
* A code editor like [VS Code](https://code.visualstudio.com/) or [Android Studio](https://developer.android.com/studio)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Mariam-mahmoud59/StoryBook.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd storybook
   ```

3. **Install dependencies:**
   ```bash
   flutter pub get
   ```

4. **Run the application:**
   ```bash
   flutter run
   ```
